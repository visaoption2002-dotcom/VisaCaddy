# Assets Directory

โฟลเดอร์สำหรับเก็บไฟล์ประกอบเว็บไซต์ VisaCaddy

## 📁 โครงสร้าง

```
assets/
├── images/     # รูปภาพ (hero, backgrounds, photos)
├── icons/      # ไอคอน (favicon, logos, UI icons)
├── fonts/      # ฟอนต์ (ถ้าใช้ local fonts)
└── videos/     # วิดีโอ (ถ้ามี)
```

## 📝 หมายเหตุ

- **images/**: สำหรับรูปภาพทั่วไป เช่น hero image, รูป background, รูปประกอบบทความ
- **icons/**: สำหรับ favicon, logo, และไอคอน UI ต่างๆ
- **fonts/**: สำหรับฟอนต์ที่โหลดจาก local (ถ้าไม่ใช้ Google Fonts)
- **videos/**: สำหรับไฟล์วิดีโอ background หรือ demo clips

## 🎨 แนะนำ

- ใช้ **WebP** หรือ **AVIF** สำหรับรูปภาพเพื่อประสิทธิภาพ
- ใช้ **SVG** สำหรับไอคอนและโลโก้
- บีบอัดไฟล์ก่อน upload
